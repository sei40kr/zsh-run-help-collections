# zsh-run-help-collections

## Install

```sh
# Load zsh-run-help-collections
zplugin light sei40kr/zsh-run-help-collections
```

## How it works

```shell-session
$ docker run # press M-h
$ # this will run
$ run-help docker-run

$ go get     # press M-h
$ # this will run
$ go help get
```
